# AppGeo

Aplicativo mobile desenvolvido com React Native (Expo) para registrar pontos geográficos com foto e localização. O usuário pode adicionar um título, descrição, capturar sua localização atual, tirar uma foto e salvar o registro em um servidor.

🚀 Funcionalidades

📍 Capturar localização do dispositivo

📸 Tirar foto com a câmera

📝 Adicionar título e descrição

💾 Salvar dados em uma API

📋 Visualizar registros cadastrados

🛠 Tecnologias

React Native

Expo

TypeScript

Expo Location

Expo Image Picker
